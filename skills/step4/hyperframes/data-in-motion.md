# Data In Motion

Data visuals must be immediately understandable.

- Numbers need visual weight: bars, counters, cards, timelines, or spatial contrast.
- Prefer one insight per segment.
- Keep at most three primary data points on screen.
- Animate values with deterministic GSAP timelines.
- Use `font-variant-numeric: tabular-nums` for counters.
- For comparison segments, use left/right or before/after structure.
- For trend segments, use height, width, opacity, or position to express direction.
